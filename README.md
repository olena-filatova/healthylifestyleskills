# Healthy Lifestyle Skills for Claude Cowork

A collection of AI skills for [Claude Cowork](https://claude.ai) focused on healthy living — meal planning, grocery ordering, nutrition, and more.

These skills were built by someone managing type 1 diabetes, so they lean towards low GI, whole foods, and blood sugar-friendly choices by default. But they're designed to be adaptable and work for any healthy diet.

---

## Skills

### 🥦 Weekly Fridge-to-Table Planner (T1D Edition)
`weekly-fridge-planner-t1d`

Checks your fridge, plans 7 days of meals, builds a shopping list, and orders groceries from your preferred online shop — all in one workflow.

→ [View skill](./weekly-fridge-planner-t1d/)

---

### 🗓️ Adaptive Weekly Menu Planner
`adaptive-weekly-menu`

Builds a realistic weekly meal plan tailored to your health conditions, schedule, cooking ability, and budget. Asks a fresh set of questions each time — location, health needs, goals, cooking time, budget — so the plan genuinely adapts week to week. Focuses on what you'll actually eat, not just what's theoretically healthy.

→ [View skill](./adaptive-weekly-menu/)

## How to install a skill

1. Download the `.skill` file from the skill's folder
2. Open Claude Cowork on your desktop
3. Drag the `.skill` file into the Cowork window, or go to Settings → Skills and install from there

**Weekly Fridge Planner** asks a few setup questions on first use, saves your preferences, and never asks again.

**Adaptive Menu Planner** asks questions every time — health conditions, schedule, goals, budget — so the plan adapts freshly each week.

---

## Philosophy

These skills are built around a few principles:

- **Whole foods first** — minimally processed, lots of vegetables, quality protein
- **Practical, not perfect** — meals that work on a busy Tuesday, not just a slow Sunday
- **Adaptable** — T1D-friendly by default, but every skill adjusts to your actual dietary needs
- **Reduces mental load** — the weekly planning and shopping cycle is exhausting; these skills handle the thinking so you don't have to
